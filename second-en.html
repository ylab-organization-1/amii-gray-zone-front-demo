There is a risk of unauthorized manipulation of routers, computers, and IoT devices
countermeasure
The solution differs depending on whether the computer's remote control function is exposed on the Internet or the router's function is exposed. Please check both settings.
- If the computer's remote control function is enabled, check the "Port Translation"* settings according to the router's manual and turn off the [port number] setting. - If the
router's remote control function (SSH) is enabled, disable the remote control function (SSH) according to the router's manual.
*The term varies depending on the manufacturer, and it may also be called "port forwarding," "port mapping," "static NAT," etc. Click here for more information.
